A propos de moi

Bonjour ! Je suis Skym, un développeur web français. J'ai toujours aimé le code et les ordinateurs en général depuis mon enfance. SkymDev

En tant que développeur web, je propose mes services sur Fiverr. N'hésitez pas à me contacter si vous souhaitez commander un site web. Je serais ravi de discuter avec vous.
Je n'utilise aucun éditeur web tel que Divi, Wordpress, etc. Tout mon code est écrit à la main, ce qui me permet d'avoir un contrôle total dessus. J'utilise des logiciels tels que Visual Studio Code, GitHub, etc. Je n'utilise que HTML et CSS. De plus, je maintiens tous les sites web que je crée avec CloudFlare.
